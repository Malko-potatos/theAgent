# Runtime Contracts

Use this folder for canonical contracts such as `TurnState`, `StepResult`, `LoopDecision`, `RuntimeEvent`, `TrajectoryEvent`, `ReplayCursor`, `ProjectionBuilder`, `TrajectoryKernel`, and `TurnReport`.

Contracts should be provider-neutral and serializable.
